# Feature Spec: Search

## 1. Overview
Provide users with the ability to quickly locate relevant technical articles based on keywords.

## 2. Goals
*   Enable efficient content discovery.
*   Surface content from titles, body text, and tags.

## 3. Scope (MVP)
*   Full-text search across all published articles.
*   Search results page displaying article title, summary, and link.
*   Basic keyword matching.

## 4. Acceptance Criteria
*   User can enter a keyword into a search field.
*   System returns a list of matching articles.
*   Search results display the article title and a short snippet.
*   Searching with no results displays a "No results found" message.
